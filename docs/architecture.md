## 模块职责

### 1. C# 桌面客户端 — 六大模块

| 子模块 | 职责 | 技术方案 |
|---|---|---|
| 截图捕获 | 框选区域截取、全屏截取、窗口截取 | 评估 Greenshot 源码；备选 Windows.Graphics.Capture API 自研 |
| 全局热键 | 注册/管理全局快捷键，处理冲突 | RegisterHotKey Win32 API，参考 Greenshot 实现 |
| 截图编辑 | 标注（箭头、矩形、文字、模糊） | 复用 Greenshot 编辑器或自研轻量版 |
| 悬浮窗 UI | 展示分析结果，支持追问输入 | WPF 无边框置顶窗口 |
| **控制台** | **配置 LLM/OCR/热键/数据，查看服务状态** | **WPF 设置窗口，从托盘菜单打开** |
| **Python 进程管理** | 启动/停止 Python 子进程，health check，自动重启 | `System.Diagnostics.Process` + 后台轮询 |

### 2. Python 推理服务 (单进程 :8900)

一个极简 FastAPI 进程，只做两件事：

| 功能 | 实现 | 说明 |
|---|---|---|
| VLM 代理 | 转发截图到 LLM API | 截图直接发给 VLM，由模型自己理解画面内容 |
| 历史存储 | JSON 文件读写 | `data/history/*.json`，零依赖 |

**设计要点**：
- Python 进程本身不加载任何 AI 模型，纯 HTTP 转发
- 截图 base64 编码后直传 VLM（视觉模型原生理解图片）
- 分析结果写入 JSON 文件，关键词搜索时匹配分析文本
- **依赖为零**：fastapi + uvicorn + httpx，总共几个包

### 3. LLM API（用户自行配置）

支持两种模式，在控制台中切换：

```
模式 A：本地模型
  用户启动 vLLM / llama.cpp / Ollama 服务 → 控制台填入地址
  例如：http://localhost:8080/v1

模式 B：云端 API
  用户填入 API Key + Endpoint
  例如：https://api.openai.com/v1
```

### 4. 控制台（配置界面）

从托盘图标右键菜单打开，配置项包括：

- **LLM 配置**：API 地址、API Key、模型名、Temperature、Max Tokens、连接测试
- **OCR 配置**：语言、GPU 开关
- **截图保存**：☑ 保存截图原图、截图保存路径
- **快捷键配置**：截图分析、纯 OCR 等操作的热键绑定
- **数据管理**：历史保留天数、数据目录、清空历史、导出备份
- **服务状态**：Python 进程状态、OCR 模型加载状态、显存占用、重启/查看日志

## 数据流

```
截图 → base64 → VLM 直接理解画面 → 分析结果文本
                                      │
                            ┌─────────┴─────────┐
                            ▼                   ▼
                     悬浮窗展示          写入 JSON 文件
                     (即时)            data/history/*.json
                                              │
       用户追问 → 关键词搜索 JSON → 上下文 → VLM → 回答
```

> VLM 是视觉模型，原生理解图片内容，无需 OCR 中转。

## 安装与启动流程

### 安装

```
AskShot-Setup.exe (NSIS 打包, ~500MB 不含VLM模型 / ~5GB 含推荐模型)
  │
  ├── 安装路径选择
  ├── 解压文件
  ├── 注册开机自启 (注册表 Run)
  ├── 注册全局热键
  └── 可选：同时安装 VLM 模型
```

### 启动

```
系统开机
  │
  ▼
AskShot.exe 自启动 → 系统托盘
  │
  ├── 检查 localhost:8900 是否存活
  │     ├── 存活 → 跳过
  │     └── 未存活 → 启动 python\python.exe services\main.py (隐藏窗口)
  │
  ├── Health check 轮询 (每 500ms, 最多 30s)
  │
  └── 托盘图标变绿 → 就绪，等待用户操作
```

### 退出

```
用户右键托盘 → 退出
  │
  ├── 杀掉 Python 子进程
  ├── 保存配置
  └── 释放资源
```
